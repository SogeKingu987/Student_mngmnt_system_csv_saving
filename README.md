# Student_mngmnt_system_csv_saving
A simple C++ Student Record Management System that demonstrates classes, vectors, multi-file project structure, and CSV file export.


Features

Add student records
View all student records
Store data using a vector<student>
Save records to a CSV file
Multi-file project structure using .cpp and .hpp files
Concepts Practiced
Classes
Vectors
Passing by reference
Const references
File handling (ofstream)
CSV file generation
Separating code into multiple source and header files

Sample CSV Output

ID,Name,Marks

1001,John Doe,85.5

1002,Jane Smith,92.0

1003,Alice Brown,78.0

Purpose

This project serves as a stepping stone toward a larger Library Management System. The primary goal was to understand how to organize code across multiple files and how to persist data by writing records to a CSV file.
